This page contains all United Nations treaty data listed at treaties.un.org. The chapter and treaty links return you to the UN treaty or chapter page and the "CSV" link downloads the treaty data. Utilities to clean and transform the data further, as well as the scripts that scraped this data can be found at http://github.com/zmjones/untreaties
